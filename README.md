# S300 — Англо-русский глоссарий технических терминов

PWA-глоссарий по электросхемам декантерной центрифуги S300.
228 терминов с переводом, контекстом и разбором словосочетаний.
Работает офлайн, устанавливается на домашний экран.

## Структура репозитория

```
index.html              — главное приложение (глоссарий)
manifest.webmanifest    — манифест PWA
sw.js                   — service worker (кэш + офлайн)
pwa/                    — иконки и favicon
```

## Локальный запуск

Так как это статический сайт с service worker, открывать через локальный
сервер (а не `file://`):

```bash
# Python
python3 -m http.server 8080
# → http://localhost:8080

# Node
npx serve .
```

## Публикация на GitHub Pages

Репозиторий уже настроен под GitHub Pages: `index.html` в корне
автоматически открывается по адресу
`https://<user>.github.io/S300/`.
